Both of the following <nav>'s are built with Chris Coyier's list-less navigation.
They are built with a <nav> > <div> > <a> layout, with an optional <nav> > <div> > <section> for submenus.
The first has a fade-in-out transition for the submenu, using a visibility and opacity trick.
The second has a slide-in-out transition, using a height and visibility trick.
All the CSS can be seen in the CSS file.
